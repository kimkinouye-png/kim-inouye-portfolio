# Kim Inouye - Portfolio

Personal portfolio website showcasing work as a product + design operations leader.

## About

This portfolio site features:
- Professional resume and experience
- Case studies and project highlights
- Sol LeWitt-inspired design aesthetic
- Dark/light mode toggle
- Responsive design
- Print-optimized resume

## Technologies

- HTML5
- CSS3 (with CSS variables for theming)
- Vanilla JavaScript
- Netlify (hosting)

## Structure

- `index.html` - Main portfolio page
- `resume.html` - Interactive resume page
- `styles.css` - Main stylesheet
- `resume.css` - Resume-specific styles
- `script.js` - JavaScript functionality (dark mode, smooth scrolling, print)
- `images/` - Image assets

## Features

- **Dark/Light Mode**: Toggle between themes with preference saved in localStorage
- **Smooth Scrolling**: Navigation links smoothly scroll to sections
- **Print Optimization**: Resume page optimized for printing/PDF export
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## Deployment

This site is deployed on Netlify with continuous deployment from this repository.
